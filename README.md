# knowsee-website

Marketing site for [knowsee](https://knowsee.co.uk) - an enterprise RAG platform.

## Stack

- Next.js 16 (App Router, static export)
- Tailwind CSS 4
- Framer Motion
- Deployed to GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` trigger GitHub Actions workflow that builds and deploys to GitHub Pages.

Custom domain: `knowsee.co.uk`
